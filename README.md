# fsociety-screen-locker
🚀 Fsociety screen locker è un'applicazione moderna sviluppata in Linguaggio Python. Pensata per creare un effetto ransomweare. Scopo principale: Scopo educativo, offre un immagine di base di fsociety, performance elevate e una configurazione rapida in locale. Open source e aperta a contributi!.
Come funziona: Estrai il file zip e avvi il file .exe nominato 'fsociety.exe'
2: Vedrai un immagine all improfvviso a schermo intero. Per toglierla basterà cliccare sulla tastiera 'ESC'
Per cambiare l immagine bisogna sostituire il file 'immagine.jpg' mi raccomando dovrà chiamarsi sempre immagine.jpg
Testata sull ultimo update di Windows 11 (versioni consigliate Windows 11,10)
